# Pearl Horizon Airlines Website
## Introduction
The sole purpose of this website is for users to book online using the website

This website is made to fulfill the requirements for Applied Business Tools and Technology course of BSHM-1F

## Requirements
The website shall have these following features:
1. Login/Signup Interface
2. Flight Management
    - Searching
    - Selection
    - History Viewing
3. Booking
    - Passenger Details
    - Seat Selection
    - Mode of Payment
    - Baggage Information
    - Meal Preference
    - Promo Code
    - Ticket Access
    - Customer Feedback
4. Administrative
    - Flight Management
    - Dashboard
    - View customer feedback
    - Booking Approval
    - Audit Logs
    - Admin Management

## Tools Used in this Project
1. Node.js - a pack
2. React.js - The main building block for this project, responsible for the frontend of the website
3. PHP - It connects the frontend to the database with what's know as API (Application Programming Interface)— a necessary building block for website building, allowing the exchange of information
